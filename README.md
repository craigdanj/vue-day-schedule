# Vue Day Schedule
View your day's schedule along a horizontal timeline. Built for VueJS.


## Install via npm
```
npm install vue-day-schedule
```



## Setup for development incase you want to contribute
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```
